# hmu-gh
> A hmu plugin for checking GitHub name availability.

This is a [`hmu`][hmu] plugin for checking if a name for a user/organization is available on GitHub.

## Installation
```shell
$ npm install -g hmu-gh
```

## Usage
```shell
$ hmu gh <...names>
```
