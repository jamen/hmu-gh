# hmu-gh
> A hmu plugin for checking GitHub name availability.

A [`hmu`](https://github.com/jamen/hmu) plugin for checking GitHub name availability.

## Installation
```shell
$ npm install -g hmu-gh
```

## Usage
```shell
$ hmu gh <...names>
```
